### :wrench: **Installation Guide**
1. **Get the Interface:** Download and install [Intiface Central](https://github.com/intiface/intiface-central/releases).
2. **Start the Engine:** Open Intiface after installing and click the giant **Play** button in the top left of app modes page.
3. **Install:** Drop [GorillaPlug.dll](https://github.com/ParaVibA/GorillaPlug/releases) into your `BepInEx/plugins` folder.
4. **Regret:** Open the menu with **F1**, hit "Reconnect Devices," and try not to get tagged. (or get tagged on purpose for more fun)
